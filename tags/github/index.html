手动解析Github Issue内容，搭建Moments

使用Curl对Github Issue API发起请求，可以得到一串Json数据。

格式：https://api.github.com/repos/USERNAME/REPONAME/issues

这些Json数据包含着可以利用的信息，我的目标是：使用Nodejs作为后端，设计一个微博、Twitter风格的前端界面，封装好简单易用的接口。

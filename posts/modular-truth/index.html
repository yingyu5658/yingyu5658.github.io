模块化真理

February 11, 2026
• 随笔

最近国家互联网信息办公室等一大堆机构，颁布了《可能影响未成年人身心健康的网络信息分类办法》，这里就不放政府网站链接了。大概浏览了一遍。有五分支持五分不爽。

让我不爽的点是，一些平台又要进行大清洗，无论是看还是发，都要麻烦些，以及文字狱可能会变得更严重。

但反过来又想了一下，我没什么好不爽的，因为墙内可能受到影响的平台似乎只有微信、QQ和BiliBili经常用。

那我只好把那五分不爽撤下，那也不代表会变成十分支持，带来的不便是一定存在的，虽然效果可能和某某专项行动一样，互联网大部分地方依旧恶臭，但多少能起到净化作用。来看看这份文件的内容。

《办法》的内容很简单，有两个条目还是很想说：

（六）通过谐音梗、缩写词、拆解字、图文结合等形式传播不良网络用语的；

从「你妹」这个听起来软萌萌的骂人用语出现，就大概可以证明，所谓的不良用语这片野草，越审查，生长得越茂盛。

（六）宣扬“读书无用论”“唯分数论”“唯升学论”等观念的；

读到这条时，我忍不住笑出来了。

在寒假前一小段时间，学校组织补……哦不对，正常上课的时候我就很想说说这个话题，但都因为太懒而搁置了，这条终于给了我动力。

这次期末考试，物理试卷很难，全班有一多半人不及格，老师安慰大家说：「就是一次考试而已，没什么大不了的！大家都没考好！」

几天前，反复强调这次期末考试的重要性的；一两个月前，年级会上反复强调中考高考重要性的；上课时，经常说「不好好读书就去地铁安检」的，也是这群老师。

「大家都没考好」这种话，简直和「你其实很聪明」可以放在一类。前者，真没考好的人一定会上当并聊以自慰，而后者可能带有激励的成分，但这种和事实方向不符的激励就是让人感到虚伪。

「大家都没考好」和内卷很像。内卷是什么状态？大家都在提升 => 大部分人回到同等水平 => 能力差的人落下得多，反之，能力强的人疯狂拔高，而属于中间水平的人，依然在苦苦挣扎，和千万牛马大军不分你我，在竞争中精疲力竭又脱不开身。

而「大家都没考好」又是一种什么状态？大家都没考好 => 大部分人回到同等水平 => 能力差的人落下得多，能力强的人成绩突出，而属于中间水平的人……正在信「大家都没考好」的鬼话。

「大家都没考好」，和「大家都考好了」的最大区别是，前者题目非常难，但总有人多变态的题目都能做得出来，我们班有考个位数的，也有年级前十的。在一个年级几百人都是如此，全市、全省、全国呢？

这就是竞争的状态。显然把学生分为能力强、中等、能力差是极其简单粗暴的，但这不是学术论文，我不会去走访调查记录数据再画个饼图，能理解竞争的状态就好。

那么，在这样难舍难分的竞争状态下，大家都在朝着一个目标厮杀，初中阶段奔高中，高中阶段奔大学，本科阶段奔研究生……一级接一级，一环扣一环，在学历的路上如此。

而大家拼命从高塔的脚底往上爬的原因，当然不是喜欢做题，一定是做题有好处。做题的好处，也不是做了题就能拿到的。费劲心思不被踹到塔底，才有那么三成的可能拿到和人竞争获得好处的机会。

然而学历不断贬值，这座塔越来越高，但向上望，永远有人。

如果一次考试不能说明什么，那塔上的人为什么不跳下来？如果一次考试真的不能说明什么，那教学楼上的人为什么突然跳下来？

唯升学论虽然极端，虽然违背所谓的社会需求、忽视人的多样性、助长功利化和攀比、脱离实践……你看，它的坏处是说不完的，但它并非不存在，并非不流行，并非能根除。

回到《办法》，反对这些思想，没有教育体制改革，教育体制也不可能改革，它选择了堵嘴、捂眼、戳聋耳朵，不容置疑。

眼不见为净。孩子们看见纸糊的蓝天白云绿草地，一定会健康成长。

我五岁的弟弟有时候不堪幼儿园的折磨，会问我，「为什么人从三岁开始都要上学哪？」

如果是我妈妈或者爸爸回答这个问题，他们通常会回答「为了在知识的海洋傲游！」「为了成长！」「为以后给祖国做贡献打基础！」

我盯着他闪着光期待着答案的眼睛：

「为了上学。」

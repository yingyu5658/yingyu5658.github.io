【Hexo】配置好服务器后如何发表编辑好的文章？

yingyu5658
2024-12-08
技术
Hexo, 教程
  1. hexo s 先在本地预览一下文章，如果都好了就进入下一步。
  2. hexo clean 清除缓存
  3. hexo g 生成文件
  4. hexo d 部署

等待部署完成后刷新页面就好了
